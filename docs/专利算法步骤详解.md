# AE-CS 专利算法步骤详解

> 一种基于一致性去噪与时空邻域保持嵌入的深度自编码器缺失数据填补方法

---

## 总览

```
输入: X ∈ R^(T×N), M^(0) ∈ {0,1}^(T×N)
      T = 时间步数, N = 变量数
      m(k,n) = 1 表示观测值存在, = 0 表示缺失

输出: X̂ (填补后的完整数据矩阵)
```

```
总损失: L_total = L_recon + λ₁·L_consist + λ₂·L_space + λ₃·L_time
```

---

## S1: 数据预处理与输入构造

**目的**: 构造模型输入，计算全局缺失率

### 1.1 观测矩阵构造
```
X^obs = X ⊙ M^(0)       # 缺失位置置零
输入 = [X^obs ‖ M^(0)]   # 沿特征维度拼接, 维度: 2N
```
- `⊙` 表示逐元素乘法 (Hadamard积)
- `‖` 表示拼接操作 (concatenation)
- 拼接后编码器同时看到"实际值"和"哪些位置有值"

### 1.2 计算缺失率 ρ
```
ρ = 1 - (1/TN) Σ_k Σ_n m(k,n)^(0)
```
- ρ 作为门控网络的先验输入
- 数据越稀疏 → ρ 越大 → 门控网络更依赖邻域编码器

### 1.3 数据归一化
- 对每个变量在观测值上计算均值和标准差
- Z-score 标准化: `x_norm = (x - mean) / std`

**对应代码**: `data/preprocessor.py` → `AECSPreprocessor`

## S2: 构建 AE-CS 网络框架

**目的**: 搭建三编码器-单解码器-门控网络的完整架构

```
┌─────────────────────────────────────────────────────┐
│                    AE-CS 架构图                      │
│                                                     │
│  [X⊙M ‖ M]──→ Encoder_orig ──→ Z_orig ─┐          │
│                                          │          │
│  [X_space ‖ M]→ Encoder_space → Z_space ─┤→ Gating → Z_fused → Decoder → X̂
│                                          │  Network  │
│  [X_time ‖ M]─→ Encoder_time ─→ Z_time ─┘   ↑      │
│                                              │      │
│                                              ρ      │
└─────────────────────────────────────────────────────┘
```

### 2.1 三个独立编码器 f_θ

每个编码器结构相同，但**权重独立**，处理不同的输入视图:

| 编码器 | 输入 | 缺失位置处理 | 作用 |
|--------|------|-------------|------|
| Encoder_orig | X⊙M (零值填充) | 置零 | 提取基础特征 |
| Encoder_space | X_space (空间KNN填充) | 保留KNN值 | 捕获空间相关性 |
| Encoder_time | X_time (时间KNN填充) | 保留KNN值 | 捕获时间趋势 |

**编码器内部结构**:
```
输入 [batch, T, 2N] → LSTM_1(128) → BN → Dropout
                    → LSTM_2(128) → BN → Dropout
                    → Dense(latent_dim, linear) → Z [batch, T, L]
```

**编码公式**:
```
Z_orig = f_θ(X ⊙ M^(0), M^(0))
       = σ(W^(2) · σ(W^(1) · [X ⊙ M^(0) ‖ M^(0)] + b^(1)) + b^(2))
```

### 2.2 解码器 g_φ

```
Z_fused [batch, T, L] → LSTM_1(128) → BN → Dropout
                      → LSTM_2(128) → BN → Dropout
                      → Dense(N) → X̂ [batch, T, N]
```

### 2.3 门控网络 (Gating Network)

```
α = softmax(W_a · [GAP(Z_orig) ‖ GAP(Z_space) ‖ GAP(Z_time) ‖ ρ] + b_a)
```

- GAP = Global Average Pooling (沿时间维度取平均)
- α = [α₁, α₂, α₃], 三个权重之和为 1

**对应代码**: `models/ae_cs.py` → `Encoder`, `Decoder`, `GatingNetwork`, `AECS`

## S3: 原始编码 → Z_orig

**目的**: 用第一编码器将观测数据映射到潜在空间

```python
# 缺失位置置零
x_zero = X * mask                           # [batch, T, N]
# 拼接掩码
x_input = concat([x_zero, mask], axis=-1)   # [batch, T, 2N]
# 编码
Z_orig = encoder_orig(x_input)              # [batch, T, L]
```

- Z_orig 是后续所有操作的基础锚点
- 一致性损失中 Z_orig 使用 stop_gradient (不接收 L_consist 的梯度)

**对应代码**: `models/ae_cs.py:537` → `self.encoder_orig(x_zero, mask, pre_filled=False)`

## S4: 多视图增强与邻域保持

这是专利的核心创新，包含三个并行子模块:

### 4.1 一致性去噪 (Coherent Denoising)

**目的**: 增强模型对缺失模式变化的鲁棒性

**步骤**:

```
对 q = 1, 2, ..., Q:
    1. 生成 Bernoulli 随机掩码:
       B^(q)_{k,n} ~ Bernoulli(1 - p_drop)

    2. 构造增强掩码:
       M^(q) = M^(0) ⊙ B^(q)     # 在已有缺失基础上额外掩盖

    3. 用 encoder_orig 编码:
       Z^(q) = f_θ_orig(X ⊙ M^(q), M^(q))

    4. 计算破坏程度权重:
       w^(q) = exp(-‖M^(q) - M^(0)‖₁ / σ_c²)
       # 破坏越重 → 权重越小
```

**一致性损失**:
```
L_consist = (1/Q) Σ_{q=1}^Q w^(q) · ‖Z^(q) - sg(Z_orig)‖_F²
```
- `sg()` = stop_gradient, Z_orig 作为锚点不接收此损失的梯度
- σ_c 自适应计算: `σ_c = sqrt(n_elements) × 0.1`

**关键参数**: p_drop=0.1, Q=3

**对应代码**: `models/losses.py` → `generate_augmented_masks()`, `consistency_loss()`

---

### 4.2 空间邻域保持 (Spatial Neighborhood Preservation)

**目的**: 在潜在空间中保持变量间的空间相似性结构

#### 第一步: 部分距离策略 (在原始数据空间 X 上)

```
对每对时间步 (t, s):
    1. 找共同观测集合:
       O_common = {n | m(t,n)=1 且 m(s,n)=1}

    2. 只在共同观测变量上计算归一化距离:
       d(t,s) = sqrt( Σ_{n∈O_common} (x(t,n) - x(s,n))² / |O_common| )

    3. 无共同观测时: d(t,s) = ∞
```

**为什么不用全部变量？** 因为缺失位置为 0，直接算距离会把"都缺失"的时间步误判为相似。

#### 第二步: KNN + 高斯核权重

```
对每个时间步 t:
    1. 找 k 个距离最近的时间步: N_t^space = argsort(d(t,:))[:k]

    2. 高斯核权重:
       w(t,s)^space = exp(-d(t,s)² / σ_s²)
       σ_s = median(有效距离)

    3. 归一化: w(t,s) = w(t,s) / Σ_s w(t,s)
```

#### 第三步: 空间 KNN 初始化 (给 Encoder_space)

```
对缺失位置 (t, n) 其中 m(t,n)=0:
    X_space(t,n) = Σ_{s∈N_t} w(t,s) · x(s,n)  # 用邻居在变量n上的观测值加权填充

对观测位置 (t, n) 其中 m(t,n)=1:
    X_space(t,n) = X(t,n)  # 保持原值
```

#### 第四步: 空间保持损失 (在潜在空间 Z 上)

```
L_space = (1/TK) Σ_t Σ_{s∈N_t^space} w(t,s)^space · ‖normalize(z_t) - normalize(z_s)‖₂²
```
- L2 归一化后计算，阻止表示坍塌到零向量

**对应代码**:
- KNN初始化: `models/ae_cs.py` → `compute_spatial_knn_init()`
- 部分距离: `models/neighborhood.py` → `find_knn_spatial_with_partial_distance()`
- 损失: `models/losses.py` → `spatial_preservation_loss()`

---

### 4.3 时间邻域保持 (Temporal Neighborhood Preservation)

**目的**: 在潜在空间中保持变量间的时间相似性结构

#### 第一步: 变量间相似度计算

```
对每对变量 (j, m):
    1. 找共同观测时间点:
       T_common = {t | m(t,j)=1 且 m(t,m)=1}

    2. 归一化距离:
       d(j,m) = sqrt( Σ_{t∈T_common} (x(t,j) - x(t,m))² / |T_common| )
```

**注意**: 空间邻域是"时间步之间"的距离，时间邻域是"变量之间"的距离

#### 第二步: KNN + 高斯核权重

```
对每个变量 j:
    N_j^time = argsort(d(j,:))[:k]     # k个最相似的变量
    w(j,m)^time = exp(-d(j,m)² / σ_t²)
    归一化: w(j,m) = w(j,m) / Σ_m w(j,m)
```

#### 第三步: 时间 KNN 初始化 (给 Encoder_time)

```
对缺失位置 (t, j) 其中 m(t,j)=0:
    X_time(t,j) = Σ_{m∈N_j} w(j,m) · x(t,m)  # 用相似变量在同一时间步的观测值填充

对观测位置: 保持原值
```

#### 第四步: 变量级嵌入映射 (三步映射)

```
(1) 变量级潜在表示:
    z_var[j] = mean_{t∈T_j} z_orig[t,:]      # T_j = 变量j被观测的时间点

(2) 变量级邻域聚合:
    z_var_agg[j] = Σ_{m∈N_j} w(j,m)^time · z_var[m]

(3) 映射回时间步:
    z_time[t] = mean_{j∈F_t} z_var_agg[j]    # F_t = 时间步t被观测的变量
```

#### 时间保持损失

```
L_time = (1/NK) Σ_j Σ_{m∈N_j^time} w(j,m)^time · ‖normalize(z_var_j) - normalize(z_var_m)‖₂²
```

**对应代码**:
- KNN初始化: `models/ae_cs.py` → `compute_temporal_knn_init()`
- 变量级映射: `models/neighborhood.py` → `compute_temporal_neighborhood_with_mapping()`
- 损失: `models/losses.py` → `temporal_preservation_loss()`

## S5: 自适应门控融合

**目的**: 根据数据稀疏程度动态分配三个编码器的权重

```
输入:
    GAP(Z_orig)  = mean_t(Z_orig)   → [batch, L]
    GAP(Z_space) = mean_t(Z_space)  → [batch, L]
    GAP(Z_time)  = mean_t(Z_time)   → [batch, L]
    ρ = 缺失率                       → [batch, 1]

拼接: h = [GAP_orig ‖ GAP_space ‖ GAP_time ‖ ρ]  → [batch, 3L+1]

注意力计算:
    h₁ = gaussian_act(W₁ · h + b₁)                → [batch, 2L]
    h₂ = gaussian_act(W₂ · h₁ + b₂)               → [batch, L]
    α  = softmax(W_α · h₂ + b_α)                   → [batch, 3]

融合:
    Z_fused = α₁ · Z_orig + α₂ · Z_space + α₃ · Z_time
```

**自适应行为**:
| 场景 | ρ | α₁ (orig) | α₂ (space) | α₃ (time) |
|------|---|-----------|------------|-----------|
| 数据完整 | 小 | 大 | 小 | 小 |
| 数据稀疏 | 大 | 小 | 大 | 大 |

**高斯激活函数**: `f(x) = 1 - exp(-x²)`, 输出范围 [0,1), 平滑可导

**对应代码**: `models/ae_cs.py` → `GatingNetwork.call()`

## S6: 解码重构与损失优化

### 6.1 解码

```
X̂ = g_φ(Z_fused)     # Decoder: [batch, T, L] → [batch, T, N]
```

### 6.2 掩码融合输出

```
训练阶段: 直接输出 X̂ (损失需要全部位置的梯度)
推理阶段: X_filled = M ⊙ X + (1-M) ⊙ X̂   # 观测位置保持原值,仅填补缺失位置
```

### 6.3 总损失

```
L_total = L_recon + λ₁·L_consist + λ₂·L_space + λ₃·L_time
```

#### L_recon: 重建损失 (仅观测位置)
```
L_recon = ‖(X - X̂) ⊙ M^(0)‖_F² / Σm(k,n)
```
- 只在 M=1 的位置计算 MSE
- 除以观测值总数进行归一化

#### L_consist: 一致性损失
```
L_consist = (1/Q) Σ_q w^(q) · ‖Z^(q) - sg(Z_orig)‖_F² / Σ_q w^(q)
```
- sg = stop_gradient (Z_orig 不接收此梯度)

#### L_space: 空间保持损失
```
L_space = mean( w_space · ‖norm(z_i) - norm(z_neighbor)‖² )
```
- L2 归一化防止坍塌

#### L_time: 时间保持损失
```
L_time = mean( w_time · ‖norm(z_var_j) - norm(z_var_neighbor)‖² )
```
- 在变量级表示上计算

### 6.4 优化

```
梯度更新: θ ← θ - lr · clip(∇_θ L_total, max_norm=1.0)
优化器:   Adam (lr=0.001)
调度器:   ReduceLROnPlateau (patience=5, factor=0.5, min_lr=1e-6)
早停:     patience=15
```

**对应代码**:
- 损失: `models/losses.py` → `total_loss()`
- 训练: `train_iterative.py` → `AECSTrainerV2.train_step()`

## 训练流程

### 自监督掩码重建

训练采用自监督掩码重建策略：在原始缺失掩码 M 基础上，额外随机掩盖 p_drop=20% 的观测值生成 M_corrupted，模型前向传播只能看到 M_corrupted 对应的数据，但重建损失在原始 M 上计算。这迫使模型从不完整信息中学会重建被额外掩盖的观测值，本质上是一种去噪自编码器（Denoising Autoencoder）机制。

```
关键: 前向传播用 M_corrupted，损失计算用 M

被额外掩盖的 20% 观测值:
  - 前向时: 模型看不到 (被当作缺失)
  - 损失时: 参与 L_recon 计算 (有真实标签)
  → 提供了自监督信号，迫使模型学到真正的重建能力
```

### 训练单步流程

```
对每个 batch (X, M):

    ┌─ 1. 自监督掩码重建 ─────────────────────────────┐
    │  B_denoise ~ Bernoulli(1 - p_drop)               │
    │  M_corrupted = M ⊙ B_denoise                     │
    │  # 在已有缺失基础上额外掩盖 20% 的观测值           │
    └──────────────────────────────────────────────────┘
                       ↓
    ┌─ 2. 前向传播 (用 M_corrupted) ───────────────────┐
    │  X_space_init = spatial_knn_init(X, M_corrupted)  │
    │  X_time_init  = temporal_knn_init(X, M_corrupted) │
    │  Z_orig  = encoder_orig(X⊙M_corrupted, M_corrupted)  │
    │  Z_space = encoder_space(X_space_init, M_corrupted)   │
    │  Z_time  = encoder_time(X_time_init, M_corrupted)     │
    │  α, Z_fused = gating(Z_orig, Z_space, Z_time, ρ)     │
    │  X̂ = decoder(Z_fused)                            │
    └──────────────────────────────────────────────────┘
                       ↓
    ┌─ 3. 一致性增强 (从原始 M 生成) ────────────────┐
    │  对 q = 1..Q:                                     │
    │      M^(q) = M ⊙ Bernoulli(1-p_consist)          │
    │      Z^(q) = encoder_orig(X⊙M^(q), M^(q))        │
    │      w^(q) = exp(-‖M^(q)-M‖₁ / σ_c²)             │
    │  注: 一致性掩码从原始M生成, 而非M_corrupted        │
    └──────────────────────────────────────────────────┘
                       ↓
    ┌─ 4. 计算损失 (用原始 M) ────────────────────────┐
    │  L_recon 在 M 上计算 (非 M_corrupted)             │
    │  → 被额外掩盖的 20% 观测值也参与 L_recon           │
    │  → 这就是自监督掩码重建的核心                       │
    │                                                   │
    │  L_total = L_recon + λ₁·L_consist                 │
    │          + λ₂·L_space + λ₃·L_time                 │
    └──────────────────────────────────────────────────┘
                       ↓
    ┌─ 5. 梯度更新 ──────────────────────────────────┐
    │  gradients = clip(∇L_total, max_norm=1.0)        │
    │  Adam.step(gradients)                             │
    └──────────────────────────────────────────────────┘
```

### 验证流程

验证与训练流程一致，同样应用自监督掩码重建，区别在于：

```
1. 使用固定随机种子 (seed=12345) 生成 corruption，保证每次验证结果可复现
2. 模型以 training=False 运行 (关闭 Dropout)
3. 不计算梯度、不更新权重
```

如果验证时不应用 corruption，则观测位置被完美复制，L_recon ≈ 0，无法评估模型真实性能。

### Epoch 级训练控制

```
对每个 epoch:
    1. 遍历所有训练 batch → 平均 train_loss
    2. 验证 → val_loss
    3. 最佳模型选择: 若 val_loss_total < best → 保存权重
    4. LR Scheduler (ReduceLROnPlateau):
       - 连续 5 个 epoch 未改善 → lr *= 0.5
       - 最小学习率: 1e-6
    5. 早停: 连续 15 个 epoch 未改善 → 终止训练
```

**对应代码**: `train_iterative.py` → `AECSTrainerV2.train_step()`, `.validate()`, `.train()`

## 推理流程

```
输入: X_test, M_test

1. 单次前向传播 (无 corruption, 无一致性计算):
   outputs = model(X_test, M_test, training=False)

2. 掩码融合:
   X_filled = M_test ⊙ X_test + (1-M_test) ⊙ X̂

3. 反归一化:
   X_final = X_filled * std + mean
```
